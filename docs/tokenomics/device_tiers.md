ShotSpec Device Tiers  
Hardware Classes, Multipliers, and Economic Roles

The device tier model ensures fairness while rewarding higher-quality data.

--------------------------------------------------------------------

Tiers

Tier 1 — Neural Node 13 TOPS  
• Baseline neural launch monitor  
• Stereo cameras  
• Battery optional  
• Good for practice and on-course  

Tier 2 — Neural Node 26 TOPS  
• Flagship device  
• Maximum multiplier  
• Supports future real-time inference upgrades  
• Best long-term earning potential  

Tier 3 — Lite Node  
• Simple camera device  
• Great for DTL or secondary angle  
• Low-cost entry point  

Tier 4 — Phone Node  
• Smartphone-only mode  
• GPS + scoring + lie + single-angle video  
• Required for every shot  

--------------------------------------------------------------------

Multipliers By Tier

Tier 1:
• 1.0× base multiplier  
• Strong impact quality  

Tier 2:
• 1.5× base multiplier  
• Superior compute & camera performance  

Tier 3:
• 0.5× base multiplier  
• Enhances Tier 1/2 when paired  

Tier 4:
• Required for attribution  
• Enables reward validation  

--------------------------------------------------------------------

Recommended Configurations

Maximum earning:
• Tier 2 Neural Node  
• Tier 3 Lite Node  
• Phone Node  
• Optional radar integration  

Budget option:
• Phone Node + Lite Node  

--------------------------------------------------------------------

Adaptive Economics

As the model improves:

• Tier multipliers decrease slowly  
• Tier 2 retains strongest long-term advantage  
• Tier 3 becomes essential at driving ranges  
• Tier 4 always required for anti-cheat  

--------------------------------------------------------------------

Vision

Every golfer owning at least one node.  
Every course eventually hosting Neural Nodes and Lite Nodes.
